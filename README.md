## 项目介绍

此项目使用 **Campus** 进行编写：<https://github.com/oddfar/campus>

> **Campus** 一款简单的后台管理系统，快速开发框架，适合大学生开发毕设，或其他小项目。

本项目为编写的一个例子：校园信息墙、表白墙、万能墙

- 在线预览：暂无

- 项目文档：<https://oddfar.github.io/campus-doc/campus-example>
